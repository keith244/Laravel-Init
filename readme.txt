# Laravel Init
Automate Laravel project setup with a simple python script

## Why this project exists
I wanted to learn more about automation using python, so to why not do it with laravel 
    (I've had a hard time in setting up laravel projects)
This project is meant to help me use python in the real world to try fix a real-pain point



## Features
- Environment validation (Checks if composer,php are installed in the system)
- Interactive folder selection (User selects where they want their project)
- Duplicate name detection. To avoid duplicates and overrides
- Automatic laravel installation

## Prerequisites
- Python 3.7+
- composer
- PHP 8.1+

## Usage
```bash
python main.py
```

Folow prompts to create project
